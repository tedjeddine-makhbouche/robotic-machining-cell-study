# robotic-machining-cell-study
Étude de cas d’ingénierie portant sur la conception d’une cellule d’usinage robotisée pour un sous-traitant automobile. Le projet traite de l’analyse fonctionnelle, du choix de solution, de l’architecture de contrôle-commande et de la modélisation GRAFCET, afin d’améliorer la flexibilité, la sécurité et la cadence de production.
